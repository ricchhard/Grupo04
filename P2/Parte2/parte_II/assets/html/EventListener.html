JavaScript HTML DOM EventListener

  • El método addEventListener() adjunta un event handler al elementos especificado.
  • El método addEventListener() adjunta un event handler a un elemento sin sobrescribir controladores de evento existentes.
  • Se pueden agregar varios event handlers a un solo elemento.
  • Se pueden agregar varios event handlers del mismo tipo a un solo elemento, como por ejemplo dos eventos de "clic".
  • Se pueden agregar event listeners a cualquier objeto DOM no solo elementos HTML, como por ejemplo el objeto de ventana.
  • El método addEventListener() hace que sea más facil de controlar como el evento reacciona al bubbling.
  • Cuando se usa el método addEventListener(), el JavaScript es separado del marcado HTML, para una mejor legibilidad y permite agregar event listerners incluso cuando no se controla ningún marcado HTML.
  • Se puede quitar fácilmente un event listener usando el método removeEventListener().

    SINTAXIS

    element.addEventListener(event, function, useCapture);

  • El primer parámetro es el tipo de evento (como "click" o "mousedown" o cualquier otro Evento DOM HTML,
  • El segundo parámetro es la función a la que queremos llamar cuando ocurre el evento.
  • El tercer parámetro es un valor booleano especificando si se usa un event bubbling o un event capturing. El parámetro es opcional.
    • Tome en cuenta que no se usa el prefijo "on" para el evento; se usa el "click" en lugar del "onclick".
