DOM Nodes

According to the W3C HTML DOM standard, everything in an HTML document is a node:
The entire document is a document node Every HTML element is an element node
The text inside HTML elements are text nodes
Every HTML attribute is an attribute node (deprecated)
All comments are comment nodes


Node Relationships

Los nodos del árbol de nodos tienen una relación jerárquica entre sí. Los términos padre, hijo y hermano se utilizan para describir las relaciones. En un árbol de nodos, el nodo superior se llama raíz (o nodo raíz) Cada nodo tiene exactamente un padre, excepto la raíz (que no tiene padre) Un nodo puede tener varios hijos.

    LA PROPIEDAD NODENAME

    La propiedad nodeName especifica el nombre de un nodo.

  • nodeName es de solo lectura
  • nodeName de un nodo de elemento es el mismo que el nombre de la etiqueta
  • nodeName de un atributo nodo es el nombre del atributo
  • nodeName de un nodo de texto es siempre #text
  • nodeName del nodo del documento es siempre #document

    • Node Type Example
    ELEMENT_NODE 1 <h1 class="heading">W3Schools</h1>
    ATTRIBUTE_NODE 2 class = "heading" (deprecated)
    TEXT_NODE 3 W3Schools
    COMMENT_NODE 8 <!-- This is a comment -->
    DOCUMENT_NODE 9 The HTML document itself (the parent of <html>)
    DOCUMENT_TYPE_NODE 10 <!Doctype html>
