DOM ANIMATIONS

La propiedad de animación de estilo permite animar las transiciones de un estilo CSS a otro estilo CSS. La animación contiene dos componentes, uno es CSS que describe el componente y otro es un conjunto de fotogramas clave que indican los estados inicial y final de los estilos.

DOM EVENTS

Los eventos se envían para notificar al código de cosas interesantes que han ocurrido. Los eventos pueden representar cualquier cosa desde las interacciones básicas del usuario para notificaciones automatizadas de las cosas que suceden en el modelo de representación.

DOM EVENTLISTENER

El método addEventListener() le permite controlar la reacción del evento correspondiente. El JavaScript está aislado del texto HTML cuando utiliza el método addEventListener(), lo que facilita su comprensión y le permite agregar detectores de eventos incluso si no controla el marcado HTML.

DOM NODE

Node es una interfaz en la cuál un número de objetos de tipo DOM API heredan. Esta interfaz permite que esos objetos sean tratados similarmente. Estas interfaces pueden retornar null en casos particulares donde los métodos y las propiedades no son relevantes. Pueden retornar una excepción - por ejemplo cuando se agregan hijos a un tipo de node del cuál no puede existir hijos.

DOM COLLECTIONS

El método getElementsByTagName() devuelve un objeto HTMLCollection. Un objeto HTMLCollection es una colección similar a una matriz de elementos HTML. Una HTMLCollection en el DOM de HTML está viva; se actualiza automáticamente cuando el documento subyacente cambia.

DOM NODE LIST

NodeList es una colección estática, lo que significa que cualquier cambio posterior en el DOM no afecta el contenido de la colección. document.querySelectorAll () devuelve un NodeList estático. Es bueno tener en cuenta esta distinción cuando elige cómo iterar sobre los elementos en NodeList, y cómo guarda en caché la longitud de la lista en particular.
